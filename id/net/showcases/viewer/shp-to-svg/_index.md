## **Contoh Langsung**

Aspose.GIS untuk .NET / C# menyajikan aplikasi ["Konverter SHP ke SVG"](https://products.aspose.app/gis/viewer/shp-to-svg) gratis online, di mana Anda dapat menyelidiki fungsionalitas dan kualitas kerjanya.

![Aplikasi Konverter SHP ke SVG](viewer.png)

## **Contoh Kode - Konverter SHP ke SVG**

Cuplikan kode berikut menunjukkan cara mengimplementasikan Konverter SHP ke SVG. Kode di bawah ini membuat peta gambar menggunakan lapisan SHP. Kemudian kita menyimpan peta ke format SVG. Jika Anda menentukan sistem referensi spasial untuk peta, semua lapisan yang dimuat akan dikonversi ke dalamnya.
Contoh tersebut menggunakan gaya yang ditentukan untuk menggambar garis. Pustaka juga memiliki gaya untuk titik, garis, dan permukaan.
