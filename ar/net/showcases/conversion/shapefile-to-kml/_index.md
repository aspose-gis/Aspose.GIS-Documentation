استخدم مكتبة Aspose.GIS for .NET لتحويل ملف Shapefile إلى تنسيق ملف KML.

## **مثال حي**

تقدم لك Aspose.GIS for .NET / C# تطبيق ["محول ملف Shapefile إلى KML"](https://products.aspose.app/gis/conversion/shapefile-to-kml) مجاني عبر الإنترنت، حيث يمكنك التحقق من الوظائف والجودة التي يعمل بها.

## **مثال للتعليمات البرمجية - محول ملف Shapefile إلى KML**

نحن ندعم Framework 4.7 أو أحدث على نظام Windows، و .NET Core 2.0 أو أحدث على نظام Windows أو Linux.
