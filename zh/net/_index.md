**欢迎使用 Aspose.GIS for .NET**

Aspose.GIS for .NET 专门设计，旨在使 .NET 开发人员能够轻松处理存储在各种文件格式中的地理空间数据。它允许您创建、读取、转换地理数据并渲染地图，而无需安装任何其他 GIS 软件。Aspose.GIS 支持 Shapefile、GeoJSON、KML、ESRI File Geodatabase 和其他格式。
