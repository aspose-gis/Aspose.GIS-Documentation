**Bem-vindo ao Aspose.GIS para .NET**

Aspose.GIS para .NET é projetado especificamente para facilitar aos desenvolvedores .NET o trabalho com dados geoespaciais armazenados em vários formatos de arquivo. Ele permite criar, ler, converter dados geográficos e renderizar mapas sem precisar instalar nenhum outro software GIS. Aspose.GIS suporta Shapefile, GeoJSON, KML, ESRI File Geodatabase e outros formatos.
