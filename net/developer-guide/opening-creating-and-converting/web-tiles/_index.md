## **Working with XYZ Tiles**
The XZY Tile ("Slippy Map") is an approach for building mapping on the web. The world map is divided into parts called tiles. All tiles are stored in Web Mapping Service tile services like Openstreetmaps, Google Hybrid, Bing, OpenCycleMap, Thunderforest, etc. And Aspose.GIS lets you work with theÂ XYZ Tiels.
