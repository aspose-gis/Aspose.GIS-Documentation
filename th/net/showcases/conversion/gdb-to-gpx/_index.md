ใช้ไลบรารี Aspose.GIS สำหรับ .NET เพื่อแปลงไฟล์จากรูปแบบ GDB ไปเป็น GPX

## **ตัวอย่างสด**

Aspose.GIS สำหรับ .NET / C# นำเสนอแอปพลิเคชัน ["GDB to GPX Converter"](https://products.aspose.app/gis/conversion/gdb-to-gpx) ฟรีออนไลน์ ที่คุณสามารถตรวจสอบฟังก์ชันการทำงานและคุณภาพได้

![ตัวแปลง GDB เป็น GPX](conversion.png)

## **ตัวอย่างโค้ด - ตัวแปลง GDB เป็น GPX**

ส่วนของโค้ดต่อไปนี้แสดงวิธีการใช้งานตัวแปลง GDB เป็น GPX หากคุณระบุระบบอ้างอิงเชิงพื้นที่สำหรับเลเยอร์ GPX ปลายทาง รูปทรงทั้งหมดจะถูกแปลงเป็นระบบนั้น

ตัวอย่างนี้แสดงความเป็นไปได้ทั่วไป Aspose.GIS สำหรับ .NET มีตัวเลือกมากมายในการ [แปลงไฟล์](https://docs.aspose.com/gis/net/vector-layers/) แต่ละรูปแบบที่ใช้เวกเตอร์จะมีตัวเลือกการอ่าน/เขียนของตัวเอง ซึ่งสืบทอดมาจากคลาส [DriverOptions](https://reference.aspose.com/gis/net/aspose.gis/driveroptions)

## **แพลตฟอร์ม - .NET / C#**

เราสนับสนุน Framework 4.7 หรือใหม่กว่าภายใต้ Windows และ .NET Core 2.0 หรือใหม่กว่าภายใต้ Windows หรือ Linux
