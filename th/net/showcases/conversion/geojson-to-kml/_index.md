ใช้ไลบรารี Aspose.GIS สำหรับ .NET เพื่อแปลง GeoJSON เป็นรูปแบบไฟล์ KML

## **ตัวอย่างสด**

Aspose.GIS สำหรับ .NET / C# นำเสนอแอปพลิเคชัน ["GeoJSON to KML Converter"](https://products.aspose.app/gis/conversion/geojson-to-kml) ฟรีออนไลน์ ที่คุณสามารถตรวจสอบฟังก์ชันการทำงานและคุณภาพในการทำงานได้

![ตัวแปลง GeoJSON เป็น KML](conversion.png)

## **ตัวอย่างโค้ด - ตัวแปลง GeoJSON เป็น KML**

สคริปต์โค้ดต่อไปนี้แสดงวิธีการใช้งานตัวแปลง GeoJSON เป็น KML หากคุณระบุระบบอ้างอิงเชิงพื้นที่สำหรับเลเยอร์ KML ปลายทาง รูปทรงทั้งหมดจะถูกแปลงเป็นระบบนั้น

## **แพลตฟอร์ม - .NET / C#**

เราสนับสนุน Framework 4.7 หรือใหม่กว่าภายใต้ Windows และ .NET Core 2.0 หรือใหม่กว่าภายใต้ Windows หรือ Linux
