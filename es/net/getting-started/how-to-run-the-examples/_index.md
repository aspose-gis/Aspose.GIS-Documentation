## **Requisitos del Software**
Asegúrese de cumplir con los siguientes requisitos antes de descargar y ejecutar los ejemplos.

1. Visual Studio 2015 o superior
1. Administrador de paquetes NuGet instalado en Visual Studio. Asegúrese de que la última versión de la API de NuGet esté instalada en Visual Studio. Para obtener detalles sobre cómo instalar el administrador de paquetes NuGet, consulte [https://docs.microsoft.com/en-us/nuget/install-nuget-client-tools](https://docs.microsoft.com/en-us/nuget/install-nuget-client-tools)
1. Vaya a Herramientas->Opciones->Administrador de paquetes NuGet->Fuentes del paquete y asegúrese de que la opción **nuget.org** esté marcada
1. El proyecto de ejemplo utiliza la función de restauración automática de paquetes NuGet, por lo tanto, debe tener una conexión activa a Internet.
## **Descarga desde GitHub**
Todos los ejemplos de Aspose.GIS para .NET están alojados en [GitHub](https://github.com/aspose-GIS/Aspose.GIS-for-.NET).

- Puede clonar el repositorio utilizando su cliente favorito de GitHub o descargar el archivo ZIP desde [aquí](https://github.com/aspose-gis/Aspose.GIS-for-.NET/archive/master.zip).
- Extraiga el contenido del archivo ZIP a cualquier carpeta en su computadora. Todos los ejemplos se encuentran en la carpeta **Ejemplos**.
- El proyecto está creado en Visual Studio 2015.
- Abra el archivo de solución en Visual Studio y compile el proyecto.
- En la primera ejecución, las dependencias se descargarán automáticamente a través de NuGet.
- La carpeta **Datos** en la carpeta raíz de **Ejemplos** contiene archivos de entrada. Es obligatorio que descargue la carpeta **Datos** junto con el proyecto de ejemplo.
- Abra RunExamples.cs, todos los ejemplos se llaman desde aquí.
- Descomente los ejemplos que desea ejecutar dentro del proyecto.

No dude en comunicarse a través de nuestros Foros si tiene algún problema al configurar o ejecutar los ejemplos.
